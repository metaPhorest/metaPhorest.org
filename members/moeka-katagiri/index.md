# 片桐萌花

2003年生まれ。早稲田大学大学院 電気・情報生命工学専攻1年、岩崎秀雄研究室在籍。2024年からmetaPhorestに参加。
幼少期より、周囲に比べて嗅覚が鋭敏であるという自覚があり、香水など身近な香りへの関心を抱いてきた。日本特有の香文化である香道の文脈や、近代以降の衛生観念の変遷に伴う脱臭の歴史に触れる中で、現代における嗅覚を用いた自己表現の在り方に疑問を抱くようになった。現在は、目に見えないが確かに存在する「体臭」が持つ情報の具体性に惹かれ、自らの身体から発せられる匂いを物質化・再構成する試みを行っている。文化史的リサーチと科学的アプローチの両側面から、嗅覚を通じた他者認識や自己の在り方を問い直し、新たな嗅覚表現の模索を続けている。
